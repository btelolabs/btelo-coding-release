### 2026-04-10

## What's New

- Dynamic Island Live Activity: shows real-time AI task progress (streaming tool name, session title, dir name) in the compact pill; expands into a completion card with app logo and checkmark when done
- In-app completion banner: when the app is in the foreground, a liquid glass card slides in from the top (mimicking Dynamic Island) — tap to navigate, auto-dismisses after a configurable duration (3s / 6s / 10s / permanent)
- Sessions initiated from macOS now also show the Dynamic Island on iPhone

## Improvements

- Completion banner dismiss duration is configurable in Notifications settings
- Bouncing dots animation smoothed to 50fps
- Refactored session data loading logic into a dedicated extension file for better code organization

### 2026-04-07

## Improvements

- Blog posts now track and display view counts with smart deduplication — repeated visits from the same IP within the same window are not counted twice
- View deduplication uses an in-memory Bloom filter (4M bits, ~400k capacity) with fill-rate-driven rotation: the filter resets only when it exceeds 30% full, checked every 6 hours, with a 5-minute double-buffer window to avoid losing counts at the boundary
- Posts with over 100,000 views bypass deduplication entirely for maximum throughput
