### 2026-04-04

## Improvements

- **Resume by ID**: Now supports session names (not just UUIDs) — handles quoted values, flag-based formats (`--resume "name"`), and bare arguments after keywords
- Improved file completion reliability
- **Empty state animation**: Refreshed with bouncing connect button, wider aspect ratio, and generic demo labels
- Removed price drop announcement popup
- Help link font size increased from caption to footnote for better readability
- **Onboarding refresh**: Redesigned onboarding pages with animated provider-specific empty state views (Browser, Files, Git, Team) and updated copy
- **Product intro sheet**: Added a swipeable product intro accessible from settings, showcasing key features with page indicators

### 2026-04-03

## What's New

- **Directory auto-complete**: File path completion now available in the chat input — type a path to get suggestions from the remote agent
- **Git graph tree view**: Enhanced git status view with tree-style directory graph rendering

## Improvements

- **Profile settings redesign**: Reorganized settings panel with clearer subscription status display and improved layout
- **Session list sort modes**: Added sorting options and multi-row layout for grouped session views
- Refined UI across browse, session, and settings views with flat minimalist + liquid glass style updates
- **File completion panel**: Typing `$` now shows a persistent panel with loading and empty states instead of briefly flashing and disappearing
- **Autocomplete panel performance**: All input panels (`/`, `@`, `$`) now use lazy rendering for faster appearance

### 2026-04-02

## What's New

- **Enhanced speech-to-text**: If the user's development environment supports it, voice input now uses a higher-quality speech recognition engine with automatic language detection — no manual language selection needed
- **Speech recognition setting**: New "Speech to Text" option in AI Provider settings to choose your preferred recognition engine
- **Auto fallback**: In Auto mode, the app automatically tries the best available engine and falls back to on-device recognition if needed

## Improvements

- Voice recording now shows a waveform capsule instead of a red circle, with a live audio level visualization
- Stopping a recording goes directly to the input box with AI-refined text — no intermediate preview step
- While AI is processing the transcript, a compact loading indicator appears in the input area with a cancel button
- The language selector is automatically hidden when using an engine that supports automatic language detection
