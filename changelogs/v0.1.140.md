## What's New

### Subscription

- **Subscription quota panel** — view your remaining usage for Claude, Codex, and Gemini providers in one place
- Quota loads on demand with skeleton screens while fetching
- Usage loads via PTY `/usage` command for accurate real-time data

### AI Providers

- **Provider controls sheet** — configure model parameters, temperature, and other settings per provider
- Added support for **Gemini**, **Codex**, and **OpenCode** providers with expanded UI controls
- Expanded Claude model list in provider settings

### Web Proxy

- **Bidirectional tunneling** — web proxy now supports full bidirectional pass-through for proxied connections
- Added auto-proxy ports toggle

### Skills

- Skills tab picker moved to navigation bar for easier access
- **Full-path skill insertion** — optionally insert the full disk path when using a skill, prefixed with "Execute this skill:" for unambiguous execution
- Select skill path based on the current AI provider automatically

### Agent Guide

- New **Agent Guide** sheet explaining how to connect and use the agent
- Localized across all supported languages
- Updated wording: "Mac" → "remote computer/machine" for clarity

### Git

- **Git stash delete** — drop individual stash entries from the stash list

### Session Improvements

- Session sort toggle with toast feedback
- Overflow count badge in session header for stacked avatars
- Confirm dialog before compacting a session to prevent accidental triggers
- Fixed truncated streaming display during live generation
- Fixed streaming message cell re-initialization for accurate display

### Bug Fixes

- Fixed: local notifications no longer fire in the background (prevents duplicates with APNs)
- Fixed: hidden files excluded from directory listings
- Improved chat bubble meta layout and input placeholder styling
