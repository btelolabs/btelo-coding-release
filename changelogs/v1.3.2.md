## Improvements

- Claude Code sessions now run inside a single long-lived process per conversation. Follow-up messages reuse the prior turns directly, so the model's recall is sharper and token costs drop sharply on long chats.
- Idle Claude processes are closed after 60 minutes, and at most 5 chats hold live processes at a time (older ones are reopened automatically when you return).

## Bug Fixes

- Switching a session's interaction mode (e.g. Trusted → Planning) now applies to the very next message, instead of silently keeping the old mode until the session was restarted.
