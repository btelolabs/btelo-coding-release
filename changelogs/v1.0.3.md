## What's New

### New Slash Commands

- **`/session-fork`** — fork the current session into a new independent tab, continuing from the same context without affecting the original session (replaces the previous `/btw` command)
- **`/copy`** — copy the last AI response or a specified message to the clipboard directly from chat

### iOS App

- Added a dedicated **Pricing** view in the app so you can review and manage your subscription without leaving the app
- Localized browse UI strings — the file browser now fully supports all 7 languages
- Fixed a bug where forked sessions could incorrectly overwrite the `ExternalID` of the parent session
- Session titles are now automatically generated when forking
- Updated the Shepherd badge icon design
