### 2026-04-06

## Improvements

- Significantly reduced CPU usage during message streaming by batching display updates (300ms throttle instead of per-chunk)
- Reduced CPU usage during text input by debouncing slash/at/file completion matching and isolating input text state from view hierarchy
- Optimized message list scrolling by merging scroll geometry handlers and eliminating per-pixel state updates
- Pre-computed tool card types to avoid repeated JSON parsing during scrolling
- Converted all 476 debug print statements to conditional logging (zero overhead in release builds)
- Reduced unnecessary SwiftUI view cascade from ConnectionService recovery tick

## Bug Fixes

- Fixed slash command panel not dismissing after Tab/Enter selection
- Fixed list flash caused by animated freshness check rebuilds when content hadn't changed
- Truncated large tool outputs (50 lines max) to prevent UI freeze when expanding tool cards
