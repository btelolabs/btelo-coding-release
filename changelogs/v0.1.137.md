## What's New

### iOS App

- **Cross-device session sync** — sessions opened on one device are now reflected across all your devices in real time
- **Mac zoom / font-size scaling** — use system zoom settings on Mac to scale the entire app UI
- **Clipboard image paste on Mac** — paste images directly from the clipboard into the chat input
- Session tabs now sort by server time for consistency
- Session resume by ID works across all AI providers
- Fixed: Apple Sign-In button uses the correct white style in dark mode

### Performance

- Sessions hash now covers only `id/title/lastMessageAt` — reduces unnecessary re-renders
