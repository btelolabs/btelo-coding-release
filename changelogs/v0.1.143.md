## What's New

### iOS App

- **Camera picker** — take a photo directly from the chat input to attach to a message
- **Directory tab row** — quickly switch between directories at the top of the sessions list
- **LRU session cache** — navigating back to a previously opened session is now instant, with no reload
- **macOS keyboard shortcuts** — use standard keyboard shortcuts (copy, paste, etc.) in the chat input on Mac
- **Smart clipboard paste** — paste text or images from the clipboard intelligently
- Session title generation now handles mixed-language content correctly
- Pro badge and subscribe button layout improvements
