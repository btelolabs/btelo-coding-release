## What's New

### Pricing & Subscription

- New **Pricing page** at `/pricing` with plan comparison and upgrade options
- Per-directory free tier limits — free users get 1 active session and 1 cron job per directory

### Session Management

- **Session fork** — create a new tab that continues from the current session context, letting you explore an alternative approach without affecting the original
- **Restore archived sessions** — previously archived sessions can be brought back from the session list
- Keyboard focus is now retained correctly after sending a message

### Connection

- New **Retry** and **Reconnect** buttons on the connection status overlay — no need to close and reopen the app after a network drop
- Improved typing indicator: compact and non-intrusive during active AI generation

### Agent

- Fixed: `--verbose` and `--stream-json` flags are now always passed correctly to Claude CLI; non-essential warnings are suppressed

### Documentation

- Updated relay documentation: "Reset Connector" is now "Switch Relay", with added self-hosted relay setup instructions
