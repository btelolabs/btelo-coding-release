## What's New

### System Tray App (macOS / Windows)

A new desktop tray application lets you control the Vibe Remote agent directly from your menu bar or system tray — no terminal required.

- **One-click launch** — start, stop, and restart the agent from the tray icon
- **Auto-download** — the tray app detects and downloads the correct agent binary for your platform automatically
- **Native terminal launch** — opens your preferred terminal (Terminal.app, iTerm2, Windows Terminal) for interactive sessions
- **Signed and notarized** — the macOS build is fully code-signed and notarized, ready for distribution

### Other Improvements

- Fixed TestFlight upload credentials configuration in the release workflow
- Improved local build pipeline for the tray app with dedicated `make tray-build` and `make tray-app` targets
