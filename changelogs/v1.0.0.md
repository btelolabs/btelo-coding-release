## What's New in 1.0

This is the first stable release of Vibe Remote. It includes a large collection of features built up over the 0.x series.

### Auto-Update

- The agent now **automatically updates itself** in the background and restarts when a new version is available — no manual `curl | sh` needed after the initial install
- Update checks run every 4 hours; restarts are deferred until the current session is idle

### Git Tree View

- New **Git Tree** tab with a fully rendered `git log --graph` view, drawn with Bézier curves on iOS
- Tap any commit to expand it inline — see changed files, additions/deletions, and diffs without leaving the view
- Added `git branch` support and directory-scoped file finding

### LAN Relay

- The agent now **auto-starts a LAN relay** in the background at launch, enabling faster local connections when your phone and computer are on the same network
- iOS app automatically discovers and offers to switch to the LAN relay
- New "Switch Relay" CTA appears in the file browser when a file is too large to preview over the cloud relay

### File Browse

- Native **file preview** for PDF, HTML, and Office/EPUB documents
- Smooth pinch-to-zoom image preview
- LAN file download with secure one-time key

### Slash Skills

- Skills now use **ETag-based conditional loading** — unchanged skills are not re-downloaded, reducing startup time
- iOS auto-updates the local skill cache in the background

### Provider Updates

- Added **Codex** (OpenAI) provider support
- Updated **Gemini** and **OpenCode** provider configurations
- Improved provider robustness: streaming failure recovery, session lifecycle handling

### Other Improvements

- CI/CD now triggers only on `git tag` push, not every commit push
- Fixed file search to include hidden directories while skipping known noise directories (`.git`, `node_modules`, etc.)
- Pre-record provider renamed for clarity
- Context recovery now includes image and file paths when switching providers
