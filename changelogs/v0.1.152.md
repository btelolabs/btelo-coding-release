## What's New

### Native File Preview

- **PDF, HTML, Office, and EPUB** files now open natively in the iOS app with full rendering
- Smooth **pinch-to-zoom** for images with momentum and snap-back physics

### Completion Alarms (Major Redesign)

- Completely redesigned alarm cards with a **content-first layout** — see the session title and result at a glance
- 4 adaptive layout variants depending on content length
- **Horizontal paging** with peek animation to swipe between multiple alarms
- New **color theme picker** with a swipeable live preview of card styles
- Monochromatic adaptive colors that look great in both light and dark mode
- Renamed to "In-App Alarm" for clarity; localized across all 7 languages
- Alarm cards use lighter gradients in light mode for better contrast
- **Dynamic Island** integration — tap the Live Activity on the Dynamic Island to dismiss the alarm
- Live Activity is now always-on during active sessions
- Removed the particle effect for a cleaner, more focused presentation
- Fixed: alarms now survive audio interruptions (phone calls, other audio sessions)

### New Slash Command

- **`/btw`** — send a side note to the AI mid-session without interrupting the current task flow

### Input & Skills

- **Input history** — swipe up/down in the text field to cycle through previously sent messages
- New alarm sounds with per-device APNs audio delivery
- Session attachments support
- Improved skills panel UI and polish

### Providers

- **Codex** (OpenAI o-series) provider added
- **Gemini** and **OpenCode** provider configuration updates
- Added comprehensive **provider robustness tests**: streaming failure recovery, session lifecycle, system prompt injection

### Git

- Added **merge conflict recovery dialog** — detects conflicts in stashed files and offers one-tap resolve
- Git discard-all now works reliably across all dirty states

### Admin

- Added **pagination** to the admin Devices tab (Connected Peers, Recent Registrations, All Connectors)

### Bug Fixes

- Fixed cooperative thread safety for `isOnWiFi` detection
- Fixed: QR scan failure no longer disconnects the current active connector
- Corrected LAN relay ping timeout to 3 seconds as specified
