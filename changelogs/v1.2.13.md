### 2026-04-15

## Bug Fixes

- On macOS, task completion now shows an in-app banner (since Live Activities are not supported on macOS)

### 2026-04-14

## Improvements

- Notifications now always use APNs, showing a banner even when the app is open — consistent with how system notifications behave
- Dynamic Island expands into a large card when a task completes, without playing an extra sound (APNs already handles the alert sound)
- If you have "Show Previews" set to Never in iOS Settings, the notification title now says "Enable notification details in Settings" as a helpful prompt
- Removed the in-app completion banner overlay — APNs handles all notifications uniformly
