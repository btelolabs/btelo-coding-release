## What's New

### Sessions

- **Session fork** — fork the current session into a new independent tab from within the app, continuing from the same context
- Direct session creation from repository cards in the git browser

### Git

- **Recent git directories** — the git browser now shows recently accessed repositories for quick access
- **Stash list pagination** — view all stash entries with pagination for large stash lists
- Optimized git root scanning with cache-inferred roots for faster directory detection

### iOS App

- **Bundled default avatars** — default avatars are now included in the app bundle with a unified URL-based cache, reducing network requests
- Subscription tier now refreshes automatically when the upgrade prompt appears; auto-dismisses if already subscribed
- Fixed: `replace_last` streaming events now correctly trigger UI updates
- Fixed: chat input editor max height is capped to prevent the toolbar from being pushed off screen
- Fixed: subscription cell updates reliably across version changes
