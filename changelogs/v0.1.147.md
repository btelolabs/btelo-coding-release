## What's New

### Persistent Daemon

- The agent now runs as a **persistent background process** — close the terminal and the agent keeps running
- The TUI becomes a detachable panel you can reattach to at any time
- Auto-generates a connector on first startup, no manual setup needed

### Live Activity

- **Live Activity on Lock Screen and Dynamic Island** — see the current AI session status without unlocking your phone
- App logo shown in the Dynamic Island during active sessions
- Live Activity updates are throttled to avoid excessive battery use
- Added toggle to enable/disable Live Activity from the app

### Auto-Update

- Agent now **checks for updates on startup** with a 4-hour cooldown — always running the latest version without manual intervention

### TUI Improvements

- Compact QR code layout with a two-column AI Providers section
- QR code always visible on dashboard; AI Providers section auto-hides when terminal is too short
- New `[o]` shortcut to open the App Store URL from the help modal
- Replaced "Reset Connector" menu item with a full Reset menu

### iOS App

- Browser tab persistence — tabs stay open when navigating away and back
- Floating browser improvements and top mode settings
- Fixed: duplicate APNs tokens are now deduplicated to prevent duplicate notifications
- Fixed: chat input bar uses solid card style on macOS to avoid glass rendering issues

### Relay

- Zero-downtime deploy with `RollingUpdate` strategy and health probes
- Added HTTP → HTTPS redirect for all relay domains
- Removed unused TURN ports to reduce attack surface

### Admin

- Geo analytics: record signup IP and country (via `CF-IPCountry` header)
