## Bug Fixes

- Fixed version number not showing in the TUI title bar due to content height being calculated one line too tall
- Fixed the last AI message sometimes appearing incomplete after task completion due to a scroll firing before layout updated
- Fixed a stale streaming rebuild task occasionally triggering a scroll that missed newly fetched messages
- Fixed an issue where two notifications (Live Activity banner + APNs) could appear simultaneously when a task completes
- Fixed a race condition where the Live Activity could get stuck in "Running" state while the APNs completion notification already showed, then triggering a second banner when the WebSocket event arrived
