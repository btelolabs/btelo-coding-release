## What's New

### Authentication

- **QR-only login** — removed manual password login; all authentication is now done by scanning a QR code for a cleaner, more secure pairing flow

### iOS App

- **Attachment support** — send files and images as part of your chat messages
- **Multi-connector** — connect to multiple agents and switch between them from the app
- Improved Dashboard UI with a cleaner layout
- Chat bubble tab icon and liquid glass scroll button
- Neutral gray chat background (removes warm tint)
- Show subscription expiry time on the profile screen
- Removed the redundant "Active" session label

### Agent & Relay

- Subscription ownership transfer — subscriptions automatically migrate to the new owner on sync instead of rejecting
- Added subscription trial tombstone to prevent abuse
- ULID format for User IDs (replacing device-based IDs)
- User account deletion — delete your account from the iOS app
- Auto-fill nickname and avatar from provider data on sign-in
- Fixed: assistant messages with empty text but non-empty content blocks are now saved correctly

### Onboarding

- Updated onboarding assets and app screenshots
