## What's New

### Performance

- **Streaming lag fix** — streaming events are now processed off the main thread, eliminating UI lag during screen recording and heavy animations
- Reliable stop button with proper process group kill for long-running tasks
- Improved streaming sync robustness with forward-compatible unknown event type handling

### Slash Skills

- **Skill picker search** — filter skills by name or description as you type
- **Recent-first sorting** — skills you've used recently appear at the top of the list
- Skills store tab extracted into its own view for cleaner navigation

### Git

- **Cross-session git summary sharing** — sessions in the same directory now share a single git status refresh, reducing redundant background work
- Reduced git refresh debounce for snappier updates after file changes

### Agent & TUI

- Non-blocking startup — the agent now starts accepting connections while still initializing background services
- Auto-refresh QR code with background update check on launch
- Updated keybindings: `r` = refresh QR / reset connector, `c` = configure via QR scan

### Website

- Replaced the hero carousel with an **auto-scrolling screenshot marquee** for a cleaner first impression
- Added an **iPhone-framed video demo** section to the homepage

### AI Thinking

- Added **thinking heartbeat** to keep the connection alive during long reasoning steps
- Unknown streaming event types are forwarded as-is for forward compatibility with new model features

### Admin

- New **Usage** tab in the admin dashboard with token consumption metrics and memory statistics
- Fixed Mac session navigation edge case

### Bug Fixes

- Git API list responses now return empty arrays instead of null when there are no results
- Removed unnecessary platform-specific branching in the browser tab
